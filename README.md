# Weather Diary (Дневник погоды)

**Автор:** Абрегова Милана  
**Вариант:** Weather Diary

## Что за программа

Программа для ведения дневника погоды. Можно добавлять записи (дата, температура, описание, были ли осадки), удалять их, фильтровать по дате или температуре.

Данные сохраняются в файл weather.json.

## Что нужно для запуска

- Python 3
- tkinter, json, datetime

## Запуск

```bash
python main.py
